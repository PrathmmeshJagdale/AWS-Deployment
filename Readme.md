# AWS Flask & Express DevOps Project

## Overview
This project demonstrates deployment of a multi-tier application using AWS services.

## Architecture
- Flask backend REST API
- Express frontend service
- AWS EC2 (single & multi instance)
- Docker, ECR, ECS (Fargate)

## Deployment Models
1. Single EC2 deployment
2. Multi EC2 distributed deployment
3. Containerized deployment using ECS

## Technologies
- Python (Flask, Gunicorn)
- Node.js (Express)
- Docker
- AWS EC2, ECR, ECS, VPC

## Future Improvements
- Nginx reverse proxy
- CI/CD using GitHub Actions
- HTTPS with ALB
